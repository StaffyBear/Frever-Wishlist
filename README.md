# Frever Wishlist

A private family gift registry PWA using HTML, CSS, JavaScript and Supabase.

## Setup

1. Upload the files to GitHub.
2. Open `js/supabase.js`.
3. Confirm your Supabase URL and anon key are correct.
4. In Supabase, create the database tables using the SQL provided in `supabase-schema.sql`.
5. Enable email/password auth in Supabase.
6. Open `index.html`.

## Main pages

- `index.html` - welcome / splash
- `login.html` - sign in
- `register.html` - create account
- `dashboard.html` - main app menu
- `registries.html` - gift registries
- `wishlist.html` - gifts within a wishlist
- `admin.html` - create registry, wishlist, and gift ideas
- `purchases.html` - purchase history
- `profile.html` - profile and sign out

## Notes

This is a working starter shell. The design and navigation are set up. The next step is connecting each button to full Supabase database actions.
